# Impact — Product Strategy

**SaaS-first growth infrastructure for professional services**

---

## Strategic Thesis

Professional services businesses—agencies, consultants, MSPs, web firms, creative studios—share one structural problem:

They sell expertise, but their growth engine is broken. Discovery is manual. Qualification is inconsistent. Nurture is neglected. Closers waste time on unqualified calls. Follow-up dies after the first "no."

Impact is the **AI-powered growth infrastructure** that fixes this. Not a CRM. Not a scraper. The operating system that owns the path from **intent to qualified booked call**.

---

## The Middle Layer Opportunity

The sales tech stack is fragmented. Each tool owns one slice:

```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   Apollo    │  │    Clay     │  │   HubSpot   │
│ Find contacts│  │ Enrich data │  │  Track CRM  │
└─────────────┘  └─────────────┘  └─────────────┘
       │                │                │
       └────────────────┼────────────────┘
                        │
                        ▼
              ┌───────────────────┐
              │      IMPACT       │
              │                   │
              │  Intent           │
              │    ↓              │
              │  Qualification    │
              │    ↓              │
              │  Nurture          │
              │    ↓              │
              │  Urgency (ethical)│
              │    ↓              │
              │  Qualified call   │
              │    ↓              │
              │  Closer brief     │
              └───────────────────┘
                        │
                        ▼
              ┌───────────────────┐
              │  Human closer     │
              │  closes deal      │
              └───────────────────┘
```

**Impact owns what no one else does:** the intelligent middle layer between raw data and human closing.

Apollo finds contacts. HubSpot tracks deals. Clay enriches records. Intercom answers website chats. Zapier connects workflows.

None of them qualify, nurture, handle objections, create ethical urgency, or prepare closer briefs. That's Impact.

## Target Market

### Primary (Phase 1–2)

Built and validated on **ninety two**—a branding agency with a defined service catalog, ICP, and sales motion.

### Secondary (Phase 3+ SaaS)

| Segment | Pain Point | Impact Value |
|---------|------------|--------------|
| **Agencies** (branding, creative, marketing) | Inconsistent BD, manual research, unqualified calls | Full-path automation from signal to closer brief |
| **Consultants** | Feast-or-famine pipeline, no nurture system | Signal-driven opportunity engine with qualification |
| **MSPs** | Long sales cycles, technical buyers hard to reach | Research + FAQ/objection handling before calls |
| **Web firms** | Commoditized pitches, low differentiation | Personalized angles from website/SEO audit signals |
| **Creative studios** | Portfolio-driven but passive inbound | Proactive signal discovery + nurture |
| **Professional services** | Partner-led sales, no SDR bench | AI SDR + qualification without headcount |

### Market Size Logic

Every agency with 5–50 employees has 1–3 people doing BD badly. Impact replaces the need for 2–5 SDRs, 1–2 researchers, and hours of unqualified calls per week. At $500–2,000/month SaaS pricing, the ROI is obvious when one additional closed deal pays for a year of subscription.

## Competitive Positioning

| Competitor / Category | What they do | What Impact does differently |
|-----------------------|--------------|------------------------------|
| **CRM** (HubSpot, Salesforce, Pipedrive) | Track deals after they exist | Create and qualify opportunities before CRM |
| **Prospecting** (Apollo, ZoomInfo) | Find contacts and lists | Find buying signals and qualify intent |
| **Enrichment** (Clay, Clearbit) | Append data fields | Synthesize research, pain points, and angles |
| **Outreach** (Outreach.io, Salesloft) | Send sequences at scale | Nurture with education; book only qualified calls |
| **AI SDR** (11x, Artisan) | Automated cold email | Signal-driven, qualified, objection-aware nurture |
| **Chat** (Intercom, Drift) | Answer website visitors | Full-path from signal to closer brief—not just chat |

**Impact's moat:** The full path. Not one step. Signal → score → research → personalize → nurture → qualify → urgency → booked call → closer brief → follow up forever.

---

## Product Principle

> **AI eliminates wasted sales time. Humans close qualified opportunities.**

This principle governs every product decision:

- If a feature puts unqualified leads on a closer's calendar, it fails.
- If a feature automates work a human shouldn't be doing, it succeeds.
- If a feature educates a prospect before pressuring them, it succeeds.
- If a feature creates false urgency, it fails.

## ninety two Offer Routing

Impact routes opportunities into configurable offers based on signal type, research, and qualification data. ninety two's initial offer catalog:

| Offer | Trigger Signals | Typical Buyer |
|-------|-----------------|---------------|
| **Website Audit / Redesign Sprint** | Website changes, poor SEO, rebrand signals | Marketing director, CEO |
| **Brand Strategy Sprint** | Rebrand, expansion, leadership change | CEO, CMO |

| **Employer Brand / Recruiting Campaign** | Hiring surges, talent competition | HR, marketing |
| **Signage + Environmental Graphics** | New locations, expansion, fleet changes | Facilities, marketing |
| **Fleet Graphics Program** | Fleet changes, rebrand, new vehicles | Operations, marketing |
| **Search Visibility Audit** | SEO signals, competitor gaps, website issues | Marketing director |
| **White-label Partner Program** | Agency partnerships, overflow capacity | Agency principals |

Offer routing is tenant-configurable. SaaS customers define their own offer catalogs in Settings.

## Go-to-Market (SaaS)

### Phase 1: Dogfood (ninety two)

Build on real workflows. Validate every engine against live opportunities. Measure: qualification rate, call quality, close rate, time saved.

### Phase 2: Design Partners (3–5 agencies)

Invite similar agencies with overlapping ICP. Configure tenant settings, offer catalogs, and nurture templates. Collect feedback on qualification grades and closer brief quality.

### Phase 3: Self-Serve SaaS

Launch `apps/platform` with onboarding wizard: ICP setup → service catalog → offer routing → integration connections → first signal import.

### Pricing Model (Draft)

| Tier | Target | Includes |

## Key Metrics (Product)

| Metric | Definition | Target |
|--------|------------|--------|
| **Qualification rate** | % of opportunities reaching A or B grade | > 40% of scored opps |
| **Call quality score** | Closer rating of brief accuracy (1–5) | > 4.0 average |
| **Unqualified call rate** | % of booked calls marked unqualified post-call | < 10% |
| **Nurture conversion** | C-grade → A/B grade within 90 days | > 15% |
| **Time to first nurture** | Signal detected → first nurture touch | < 24 hours |
| **Closer prep time** | Minutes spent preparing before a call | < 5 minutes |
| **Signal-to-close cycle** | Days from signal to won deal | Decreasing quarter-over-quarter |

## What Impact Will Never Be

- A CRM (HubSpot/Salesforce remain system of record for closed deals)
- A lead scraper or list builder
- A cold email blaster
- A chatbot replacement for Intercom
- A project management tool
- A tool that creates false urgency or manipulates prospects

Impact feeds existing tools with qualified, nurtured, briefed opportunities. It does not replace them.
